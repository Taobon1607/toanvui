# Ứng dụng Học Toán cho trẻ em (Math Learning Web App)

## 1. Khởi tạo dự án
- [ ] Khởi tạo dự án với HTML, CSS Vanilla và JavaScript thuần (Do máy chưa cài Node.js).
- [ ] Xây dựng cấu trúc thư mục phổ thông (js, css, assets, data) và components js độc lập.

## 2. Thiết kế dữ liệu mẫu (Mock Data)
- [ ] Dữ liệu các Lớp học (Lớp 1, 2, 3...)
- [ ] Dữ liệu các Chủ đề toán học (Phép cộng, Phép trừ, Hình học...)
- [ ] Dữ liệu Bài tập (Câu hỏi, đáp án, hướng dẫn giải chi tiết từng bước)
- [ ] Dữ liệu Kiến thức (Các khái niệm toán học liên kết từ bài giải)

## 3. Xây dựng giao diện (UI) - CSS thuần
- [ ] Thiết lập hệ thống CSS (biến màu sắc rực rỡ, typography rõ ràng, animation) thân thiện với trẻ em.
- [ ] Component: Header (Điều hướng chung)
- [ ] Page: Trang chủ (Chọn Lớp/Trình độ)
- [ ] Page: Danh sách chủ đề toán theo lớp
- [ ] Page: Giao diện giải toán (Đọc câu hỏi, Chọn/Nhập đáp án)
- [ ] Component: Hiển thị hướng dẫn giải từng bước và liên kết kiến thức bằng tooltip hoặc modal.

## 4. Hoàn thiện và Kiểm tra
- [ ] Thêm hiệu ứng âm thanh/chúc mừng (Visual effects) khi giải đúng bằng CSS animations và JS thuần.
- [ ] Kiểm tra responsive trên các thiết bị.
- [ ] Lấy review từ người dùng.
